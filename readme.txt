1. Open the index.html file located in the email_signature directory.
2. Click `Copy as HTML` button after editing the content properly.
3. Go to https://html-online.com/editor/ and press Ctrl + V inside the right panel.
   Then you can notice the left panel populated with the html elements just the same as the you saw in the html file.
4. In the left panel, please select all the elements and press Ctrl + C (copy)
5. And then go to Settings -> Signature in gmail settings and click `Create New` to create a email signature.
6. Then focus on the editing panel of email signature, press Ctrl + V to paste the copied content.
7. And choose the Signature defaults. That's all.
